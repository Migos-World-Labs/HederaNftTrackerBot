/v1/public/market/activity
Get the latest activity from the market.

This endpoint returns the latest market activity, including sales and listings. You can filter by token address, serial ID, activity type, and more.

Parameters
Name	Description
apikey *
string
(query)
Your API key (generated from the settings panel on sentx.io).

apikey
token
string
(query)
Leave blank to retrieve all activity, or pass the non-fungible token address to filter by token.

token
serial
string
(query)
Leave blank to retrieve all activity, or pass the serial ID of the token (if token is specified).

serial
activityFilter
string
(query)
Leave blank to retrieve all activity, or pass one of these options: 'Sales', 'Listings', 'All'.

activityFilter
amount
integer
(query)
Number of records to return (default: 50).

Default value : 50

50
page
integer
(query)
The page to be returned (default: 1).

Default value : 1

1
includePreviouslyBoughtPrice
boolean
(query)
Pass 1 if you want to include the previous price that the seller bought the item for before this sale.


--
hbarMarketOnly
boolean
(query)
Pass 1 to filter and show only HBAR market feed, or leave blank to query all sales across HTS markets.


--
paymentTokenAddress
string
(query)
Pass a token address to filter and show only that specific token address feed.

paymentTokenAddress
